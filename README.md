1. 利用者情報の取得と利用目的 本アプリ（LocationTimeLine）は、以下の情報を取得し、それぞれの目的のために利用します。

位置情報（GPS）

利用目的: 移動経路の記録、地図への表示、および走行速度・距離の計算を行うため。

詳細: 本アプリは、移動ログを記録するためにバックグラウンドでも位置情報を取得します。取得した位置情報データは、アプリ内（端末ローカル）およびペアリングされたApple Watchとの通信のみに使用され、開発者のサーバーへ送信・保存されることはありません。

端末情報・識別子

利用目的: 広告配信およびアプリ内課金の管理のため。

2. 第三者モジュールの利用 本アプリは、広告配信および利用状況解析、課金管理のために以下の第三者サービスを利用しています。これらのサービスは、利用者情報を自動的に取得する場合があります。

Google AdMob（Google Inc.）

広告配信のために利用されます。

RevenueCat Privacy Policy
