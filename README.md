#### LocationTimeLine プライバシーポリシー

本アプリケーション（以下「本アプリ」といいます）は、利用者情報、広告、iCloudなどの外部サービス連携において、以下の通りプライバシーポリシーを定めます。

---

### 1. 収集する利用者情報および収集方法

本アプリは、以下の利用者情報を取得し、利用します。

| 収集する情報 | 収集方法 | 利用目的 |
| :--- | :--- | :--- |
| **正確な位置情報 (GPSデータ)** | アプリ内での計測開始時（バックグラウンド含む） | 走行ルートの記録、地図上での表示、リアルタイムの速度計測、及び履歴としての保存のため。 |
| **広告識別子 (IDFA)** | Google AdMob SDKを通じた自動取得 | ユーザーに合わせた広告表示、および広告効果の測定のため。**（※ATTフレームワークにて許可を得た場合のみ）** |
| **デバイス識別子（UUID等）** | 自動取得 | アプリ内課金（Pro版）の購入状態を識別・管理するため。 |
| **購入履歴** | Apple StoreKitおよびRevenueCat SDKを通じた自動取得 | Pro版の機能提供、購入復元、およびサブスクリプションの有効性確認のため。 |
| **クラッシュログ** | アプリ内でのエラー発生時 | アプリの不具合を解析し、品質を向上させるため。 |

---

### 2. 外部サービスとの連携について

本アプリは、以下の外部サービスと連携して動作します。

#### A. iCloud / CloudKit (SwiftData)
* **データの保存先:** 利用者自身のiCloudアカウントのプライベートデータベース。
* **開発者のアクセス:** 開発者は、利用者データの**内容を閲覧、編集、削除することはできません。**データの管理権限は利用者に帰属します。
* **利用目的:** 異なる端末間での計測履歴の同期、およびバックアップのため。

#### B. Google AdMob (広告)
* 本アプリは広告配信のため Google AdMob を利用しています。
* AdMob は、広告識別子（IDFA）やデバイス情報に基づき、パーソナライズされた広告を配信する場合があります。
* 広告配信における利用者情報の取り扱いについては、[Googleのプライバシーポリシー](https://policies.google.com/privacy)をご確認ください。

---

### 3. 情報の利用目的

収集した情報は、主に以下の目的で利用します。

1.  本アプリの各種機能（位置記録、速度表示、履歴管理など）を提供するため。
2.  アプリ内課金（Pro版）の提供および利用資格を確認するため。
3.  アプリの不具合やクラッシュの原因を解析し、製品の品質向上に役立てるため。
4.  パーソナライズされた広告を提供するため（ユーザーの同意がある場合）。

---

### 4. 利用者情報の管理

* 収集した利用者情報は、法令に基づき厳重に管理し、漏洩、紛失、破壊、改ざんを防止するための適切な措置を講じます。
* 法令に定める場合を除き、利用者の同意なく第三者に個人情報を提供することはありません。

---

### 5. プライバシーポリシーの変更

本ポリシーの内容は、法令の改正、本アプリの機能変更に伴い、利用者の承諾なく変更されることがあります。変更後のプライバシーポリシーは、別途定める場合を除き、本アプリに表示された時点から効力を生じるものとします。
